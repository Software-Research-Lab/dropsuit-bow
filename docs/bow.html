Class: bow

bow(inputopt, s_dostemopt, d_dostemopt) → {object}

new bow(inputopt, s_dostemopt, d_dostemopt) → {object}

Creates a new instance of the 'Bag of Words' (bow) processing object. This function can process both an input sentence/word, or use the default input specified in the constructor. Processes default object instance key value (req_arr: requests).
Parameters:
Name Type Attributes Default Description
input string <optional>
 null The input sentence/word to be processed. Leave as null to use the default input specified in the constructor.
s_dostem boolean <optional>
 false Enable or disable stemming for input string tokens.
d_dostem boolean <optional>
 false Enable or disable stemming for data tokens.
